كلمات وحروف
============

مولّد كلمات مصمم خصيصا لمسابقات الحروف المعجمية مثل : شقلبة، زائد حرف، تغيير حرف.. وذلك باعتماد كلمة مدرجة

تم تطوير النسخة التجريبية الأولى اعتمادا على معجم الوسيط وذلك بتحويل الكتاب إلى قاعدة بيانات
